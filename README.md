## 上传 GitHub
把以下文件全部上传到仓库根目录：

```text
index.html
fish-loop-1080p.mp4
fish-loop-720p.mp4
fish-poster-hq.jpg
.nojekyll
.gitignore
README.md
```

然后在 GitHub 仓库中：
1. Settings
2. Pages
3. Source 选择 Deploy from a branch
4. Branch 选择 main
5. Folder 选择 / (root)
6. Save

生成 HTTPS 链接后即可发给别人打开。
